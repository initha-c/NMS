# Network Monitoring System

## Overview

This project implements a simplified Network Monitoring System using Java 21, Spring Boot, Kafka, H2 Database, and Spring Cloud Gateway.

The implementation is split into separate microservices:

1. RouterSimulator
2. SimpleNMS
3. API Gateway

Each service is maintained as a separate Spring Boot application.
